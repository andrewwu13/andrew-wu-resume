## Building the Resume Locally
If you'd like to generate the resume yourself:

`
git clone https://github.com/andrewwu13/andrew-wu-resume
`

`
cd andrew-wu-resume
pip install rendercv
`

`
rendercv render
`

This will generate a PDF using the configuration defined in the YAML files.

## About RenderCV 
RenderCV is an open-source resume framework that uses structured data (YAML) to generate beautifully formatted PDFs.

Learn more here: https://rendercv.com/
